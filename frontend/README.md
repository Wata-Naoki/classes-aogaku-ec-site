# フロントエンド

## ローカル環境

パッケージのインストール
```bash
npm install
```

画面起動
```bash
npm start
```

## 環境変数の設定

環境変数準備
```bash
cp .env.example .env
```

```bash
REACT_APP_ENDPOINT=
REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_AUTH_DOMAIN=
REACT_APP_FIREBASE_PROJECT_ID=
REACT_APP_FIREBASE_STORAGE_BUCKET=
REACT_APP_FIREBASE_MESSAGE_SENDER_ID=
REACT_APP_FIREBASE_APP_ID=
```

## ローカルのURL
```bash
http://localhost:3000
```
